flat()

將多維陣列扁平化為一維陣列

let arrA = [1,2,[3],[4,[5,[6]]]]

arrB = arrA.flat()
arrC = arrA.flat(2)
arrE = arrA.flat(1)
arrD = arrA.flat(Infinity)

shift()

從陣列前端取出第一個元素值並回傳，會修改原陣列

let shape = ["box", "ovel", "line"]

let cutShape = shape.shift()

unshift()

將元素新增至陣列的開頭，以「,」接續複數值，會修改原陣列、並回傳陣列的新長度

let color = ['white', 'yellow', 'black']

let soul = ['people']

color.unshift('red')
soul.unshift('god','devil')

fill()

從指定位置，將該範圍內的陣列元素、取代為指定值

let arrNum = [1,2,3,4]

arrNum.fill('a',2,5)
arrNum.fill('b',1)
arrNum.fill('c')

splice(index[,delnum[,additem]])

插入、刪除、替換陣列內元素，會修改原陣列
  • index－表示開始的位置
  • delnum－表示是否刪除
    • 可省略，表示刪除 index 後所有的元素
    • 若為 0 或負數，表示不刪除任何元素
  • additem－要替換的值()

let month = ['Jan', 'March', 'April', 'June']

month.splice(1, 0, 'Feb')
month.splice(4, 1, 'May')

＊窩不想改變原陣列捏？

toSpliced

一樣的用法，但將結果另存副本

=> 請觀察 splice()、toSpliced() 的差異

splice()
let Array = [1, 2, 3, 4]
let arrSplice = Array.splice(1,2)
toSpliced()
let Array = [1, 2, 3, 4]
let arrToSpliced = Array.toSpliced(1,2)

join([separator])

將陣列裡的每筆元素、用符號串成字串

let emotion = ['love','happy','angry','sad']

let emotionJoin = emotion.join('-')

slice(start,end)

從索引值 start 到索引值 end ，之間複製出部分陣列，不會更改原本陣列
  • arr.slice()
  • arr.slice(start)
  • arr.slice(start,end)

let zooAustralia = ['kangaroo', 'wombat', 'koala', 'glider', 'kookaburra']

zooAustralia.slice()
zooAustralia.slice(2,4)
zooAustralia.slice(1,5)
zooAustralia.slice(-2)
zooAustralia.slice(2,-1)
